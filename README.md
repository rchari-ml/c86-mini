# c8-labs

A composition of artifacts to assist with running Camunda 8.7 (or higher) platform in GKE cluster.

Aims to support running multiple parallel deployments.

For any questions or feedback, write to contact@makelabs.in

Pre-requisites to join the program
Gmail account
Project Lens UI app (Free version)  -> https://k8slens.dev
GCP Cloud SDK (gcloud CLI)          -> https://cloud.google.com/sdk/docs/install
