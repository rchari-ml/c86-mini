Camunda 8.6 (or higher) cluster is setup on Kubernetes cluster running on Ubuntu Pro. The cluster nodes are managed by Rancher Desktop by SUSE. The setup includes below components.

  • Elastic and Kibana
  • Prometheus and Grafana
